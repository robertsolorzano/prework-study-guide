HTML

  • Anchor tag (a): to link URLS on webpage
  • Break tag (br): Creates break in line/empty space
  • Image tag (img): Parent element containing info on the image
  • src: attribute that defines location/path for image
  • Unordered List tag (ul): Creates bullet point lists
  • Ordered List tag (ol): Creates numbered lists
  • List tag (li): contains list info
  • href: attribute that specifies location of external URL
  • rel: attribute that specifies relationship between doc and linked doc

CSS

  • Box Model: Content, Padding, Border, Margin
  • "*" aka Astrik: Is a Selector that applies values to entire HTML
  • Declarations: Property to Value
  • A Margin indicates how much space we want around the content inside an element
  • A Padding indicates how much space we want around the content inside an element

Git

  • git status: Checks what branch we are currently in
  • git checkpoint -b "insert branch path name here": Creates a new branch/switches to it
  • git pull origin main: Pulls latest main branch in folder
  • ls: Creates list in directory
  • cd: Changes directory (add '..' to return back)
  • start: Opens file
  • code .: Opens folder in VS Code

Javascript

  • console.log(): Method to log variable to web console
  • equal operator in JS contains 3 "==="
  • unequal operator in JS contains "!==="
  • Calling JS file uses script tag followed by a src attribute
  • var defines variable
