Prework Study Guide

HTML

  • Anchor tag (a): to link URLS on webpage
  • Break tag (br): Creates break in line/empty space
  • Image tag (img): Parent element containing info on the image
  • src: defines location/path for image
  • Unordered List tag (ul): Creates bullet point lists
  • Ordered List tag (ol): Creates numbered lists
  • List tag (li): contains list info

  • cd: Changes directory (add '..' to return back)
  • start: Opens file
